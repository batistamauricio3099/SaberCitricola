# SaberCitrícola 🍋

**Knowledge Management System (KMS) for the Citrus Industry**

This repository contains the source code for my final degree project as a **Systems Engineer**. **SaberCitrícola** is a platform designed to centralize, preserve, and distribute technical and strategic knowledge within the citrus sector in Tucumán, Argentina.

---

### 📋 Overview
The citrus industry often faces knowledge silos. This project provides a structured solution to:
* **Centralize Knowledge:** Store technical research, crop management protocols, and market insights.
* **Collaboration:** Enable experts and producers to share experiences and best practices.
* **Decision Making:** Provide a dashboard for data-driven insights in the citrus production chain.

### 🛠️ Tech Stack
* **Language:** C#
* **Framework:** .NET / ASP.NET
* **Architecture:** Layered Architecture (N-Tier) for scalability and maintainability.
* **Database:** SQL Server
* **Domain focus:** Knowledge Management Systems (KMS).

### 🚀 Key Features
* **Semantic Search:** Efficiently find documents and technical guides.
* **Knowledge Life Cycle:** Modules for capture, storage, and dissemination of information.
* **Role-Based Access Control:** Secure management for different stakeholders in the industry.
